# **ZavodMevBot Guide** (Automated Solana On-Chain Arbitrage Bot)

## Getting Started

### Step:1 Get a private RPC

Make sure you find a private RPC, public ones won't work for this bot.

### Step:2 Get a linux machine

The bot runs on linux, so you need to either rent one or use your own. The bot has very minimum requirement for hardware. In theory you can run it on any linux machine.

### Step:3 Download and unzip

Download **ZavodMevBot** linux version from the official site and unzip.

### Step:4 Config and Run.

Refer to config.toml.example file to create a config file. You should create a config file called config.toml.

Last step, give permissions to the bot to run

```
chmod +x *
```

Now you are all set to run!

To run the bot, simply do the following:
```
./zavod-mev-bot-rust-version-cli run config.toml
```
